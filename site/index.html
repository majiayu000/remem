The problem

Every new session starts from zero.

Context you spent hours building disappears the moment the thread ends.

How it works

Hooks into your agents. Nothing to capture by hand.

remem records raw session evidence, distills it in the background, then serves compact memory back through hooks and MCP.

Search architecture

Four retrieval channels, fused.

A query is routed through BM25, entities, temporal parsing, and LIKE fallback, then merged with Reciprocal Rank Fusion.

  • Project-scoped search prevents cross-project leakage.
  • CJK segmentation and Chinese-English synonym expansion improve recall.
  • Title-weighted BM25 and topic-key dedup keep results compact.

RRF = sum 1 / (60 + rank)

Install

Use the same commands as the README.

Start with Homebrew, the prebuilt GitHub Release installer, or Cargo. Then restart your coding agent.

curl -fsSL https://raw.githubusercontent.com/majiayu000/remem/main/install.sh | sh

Secure by default

  • SQLCipher encryption at rest.
  • Data directory 0700 and key file 0600.
  • REST API binds to 127.0.0.1 and requires a local token.
  • Uninstall removes hooks and MCP config without deleting memory data.
